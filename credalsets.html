Credal Sets¶

Class definition & methods¶

class murasyp.credalsets.CredalSet(data=set([]))[source]¶

A mutable set of probability mass functions

param:a Set of ProbMassFunc or an Set (such as a dict); in the latter case, a relative vacuous credal set is generated.
type:MutableSet

Members behave like any set, Moreover, they can be conditioned just as ProbMassFunc are and lower and upper expectations can be calculated, using the * and ** operators, respectively.

>>> p = ProbMassFunc({'a': .03, 'b': .07, 'c': .9})
>>> q = ProbMassFunc({'a': .07, 'b': .03, 'c': .9})
>>> K = CredalSet({p, q})
>>> K
CredalSet(set([ProbMassFunc({'a': '7/100', 'c': '9/10', 'b': '3/100'}), ProbMassFunc({'a': '3/100', 'c': '9/10', 'b': '7/100'})]))
>>> f = Gamble({'a': -1, 'b': 1})
>>> K * f
Fraction(-1, 25)
>>> K ** f
Fraction(1, 25)
>>> A = {'a','b'}
>>> (K | A) * f
Fraction(-2, 5)
>>> (K | A) ** f
Fraction(2, 5)
add(p)[source]¶

Add a probability mass function to the credal set

param:a mapping (such as a dict) to nonnegative Rational values, i.e., Fraction. The fractions may be specified by giving an int, a float or in their str-representation. Sum-normalization is applied.
type:Mapping 
>>> K = CredalSet()
>>> K
CredalSet(set([]))
>>> p = ProbMassFunc({'a': .06, 'b': .14, 'c': 1.8, 'd': 0})
>>> K.add(p)
>>> K
CredalSet(set([ProbMassFunc({'a': '3/100', 'c': '9/10', 'b': '7/100'})]))

Remove a probability mass function from the credal set

param:a probability mass function
type:ProbMassFunc

CredalSet(set([ProbMassFunc({'a': 1}), ProbMassFunc({'b': 1})]))

>>> K
CredalSet(set([ProbMassFunc({'b': 1})]))
