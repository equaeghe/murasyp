Sets of desirable gambles¶

class murasyp.desirs.DesirSet(data=set([]))[source]¶

A mutable set of rays

type data:a Set of Mapping, each of which can be normalized to a Ray, i.e., it must be nonconstant.

Features:

  • There is an alternate constructor. If data is a Set without Mapping-members, then a relative set of desirable gambles is generated.

    >>> DesirSet(set('abc'))
DesirSet(set([Ray({'a': 1}), Ray({'b': 1}), Ray({'c': 1})]))

discard_redundant()[source]¶

Remove redundant elements from the set of desirable gambles

Redundant elements are those that are not extreme rays of set of desirable gambles’s convex conical hull.

>>> D = DesirSet(set('abc'))
>>> D.add({'a': 1, 'b': 1, 'c': 1})
>>> D
DesirSet(set([Ray({'a': 1, 'c': 1, 'b': 1}), ...]))
>>> D.discard_redundant()
>>> D
DesirSet(set([Ray({'a': 1}), Ray({'b': 1}), Ray({'c': 1})]))
set_lower_pr(data, val)[source]¶

Set the lower probability/prevision (expectation) of an event/gamble

arg data:the gamble for which a probability/prevision value is given
type data:arguments accepted by the Ray constructor
arg val:the probability/prevision value
type val:a representation of Real

The marginal gamble corresponing to the prevision specification is calculated and added to the set of desirable gambles.

>>> D = DesirSet()
>>> D.set_lower_pr(Gamble({'a', 'b'}) | {'a', 'b', 'c'}, .4)
>>> D
DesirSet(set([Ray({'a': 1, 'c': '-2/3', 'b': 1})]))

Note

The domain of the input gamble determines the conditioning event.
