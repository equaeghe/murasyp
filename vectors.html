class murasyp.vectors.Vector(mapping)[source]¶

Vectors are immutable, hashable rational-valued functions

param:a mapping (such as a dict) to Rational values, i.e., Fraction. The fractions may be specified by giving an int, a float or in their str-representation.
type:Mapping

This class derives from Function, so its methods apply here as well. This class’s members are also hashable, which means they can be used as keys (in Set and Mapping, and their built-in variants set and dict). What has changed:

  • unspecified values are assumed to be zero;
  • the union of domains is used under pointwise operations;
  • a vector’s domain can be restricted/extended to a specified Set.
>>> f = Vector({'a': 1.1, 'b': '-1/2','c': 0})
>>> f
Vector({'a': '11/10', 'c': 0, 'b': '-1/2'})
>>> f['d']
0
>>> g = Vector({'b': '.6', 'c': -2, 'd': 0.0})
>>> (.3 * f - g) / 2
Vector({'a': '33/200', 'c': 1, 'b': '-3/8', 'd': 0})
>>> f | {'a','b'}
Vector({'a': '11/10', 'b': '-1/2'})
>>> f | {'a','d'}
Vector({'a': '11/10', 'd': 0})

sum_normalized()[source]¶

‘Sum-of-values’-normalized version of the vector

returns:the gamble, but with its values divided by the sum of the vector’s values
rtype:Vector 
>>> Vector({'a': 1, 'b': '-1/2','c': 0}).sum_normalized()
Vector({'a': 2, 'c': 0, 'b': -1})
>>> Vector({'a': 1, 'b': -1,'c': 0}).sum_normalized() == None
True

Note

None is returned in case the the sum of the vector’s values is zero.
